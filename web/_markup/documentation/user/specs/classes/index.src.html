Вспомогательные классы js

Общие вспомогательные классы js

  • js-popup-opener - указывается на элементе, который открывает попап. Обязательно должен быть указан data-popup-opener="#%popupId%"
  • js-popup-closer - указывается для закрытия попапа
  • js-popover-opener - указывается на элементе, который открывает поповер. Обязательно должен быть указан data-popover-opener="#%popoverId%"
  • js-helper-opener - указывается для открытия хелпера, обязательно дожен быть указан data-text="Текст хелпера"
  • js-scrollable - указывается для добавления скролла в блок. Еще нужно указать максимально допустимую высоту блока в стилях.
  • js-styled-checkbox - указывается на элементе чекбокса для стлизации
  • js-calc-height - указывается для подсчета высоты блока, чтобы он дотягивался к футеру. Указывается на контенте страницы или на блоках, которые нужно дотягивать.
  • js-calc-height-private-menu - указывается для подсчета высоты меню в личном кабинете
  • js-ajax-form-submit - указывается на форме для аякс отправки данных
  • js-fixed - указывается для фиксации блока главного поиска
  • js-fixed-side-banner - указывается для фиксации боковых баннеров
